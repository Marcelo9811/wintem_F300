Arquitectura

WINTEM.txt → Python local → docs/ (HTML + PNG + CSV) → GitHub Pages

El navegador no ejecuta el análisis. Tkinter, Matplotlib y Cartopy solo se utilizan al regenerar los productos en el computador del analista.

Actualizar los resultados

  1. Ejecute build_static_site.ps1 "C:\ruta\WINTEM.txt".
  2. Abra docs/index.html y revise figuras, conteos y CSV.
  3. Confirme y suba a GitHub los cambios de la carpeta docs/.

Activar Pages una sola vez

  1. Abra Settings → Pages en el repositorio.
  2. En Build and deployment, seleccione GitHub Actions.
  3. El workflow deploy-pages.yml copiará docs/ al alojamiento.

Ese workflow no instala ni ejecuta Python: solo publica archivos ya calculados.
